# Templates

This directory contains formatting templates for final CV output.

## Available Templates
- (Future) LaTeX template for ATS-friendly PDF generation
- (Future) Word/DOCX template for recruiter-friendly formats

## Current Approach
During the initial phase, the system generates CVs in **Markdown format** which can be:
1. Converted to PDF via any Markdown-to-PDF tool
2. Pasted into Google Docs / Word for final formatting
3. Fed into a LaTeX pipeline for professional output

## Adding Templates
Place new templates in this directory. Reference them from the `resume-tailor` skill.
