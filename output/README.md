# Output Directory

This directory contains the generated artifacts from each CV generation run.

## Files Generated During a Run
- `job_analysis.md` — Structured breakdown of the target job requirements
- `match_report.md` — How well the candidate matches the job
- `cv_draft.md` — The tailored CV draft
- `validation_report.md` — ATS compliance validation results

## Archive
The `archive/` subdirectory contains saved outputs from previous runs, organized by company name and date.

> **Note**: Output files are overwritten with each new run. Use the archive feature to preserve past results.
